### Чек-листы
- **259 пунктов** из файла «Чек-листы ДПБ 2026.xlsx»
- Каждый пункт содержит точную ссылку на НПА РК (Приказы МЧС, ТК РК, Распоряжения АО)
- 8 разделов: Производственные территории, Здания, Технологические помещения,
  Санитарно-бытовые помещения, Котельные и резервуары, Транспорт, Документация, Бригады
- **Иные несоответствия** — свободный ввод нарушений, не отражённых в чек-листе
  (наименование, описание, ссылка на НПА, добавление/редактирование/удаление)

### География
- **6 регионов:** Алматинский, Восточный, Западный, Северный, Центральный, Южный
- **23 области и города:** все регионы РК, включая г. Астана и Акмолинская обл.
  (Центральный регион), Павлодарская обл. (Восточный регион)
- Филиал АО «Казахтелеком» подставляется автоматически по выбранной области

### Указание по производственной безопасности (PDF)
- Генерируется автоматически после завершения проверки
- Содержит: шапку с реквизитами (регион, область, город, адрес, филиал, подразделение)
- Ф.И.О. инспектора и ответственного на объекте, дата проверки
- Таблицу всех выявленных нарушений (по чек-листу + иные) с НПА и сроками устранения
- Полную таблицу всех пунктов проверки с ответами Да/Нет/Н/П
- Блок подписей: инспектор + ответственный на объекте
- Открывает диалог печати браузера → «Сохранить как PDF»

### Маршрут согласования
1. Инспектор заполняет указание
2. Направляет ответственному руководителю (Push + Email)
3. Руководитель принимает и назначает исполнителей
4. Исполнители устраняют нарушения с фото-подтверждением
5. Инспектор закрывает указание

### Дашборд аналитики (4 вкладки)
- **Обзор:** KPI открытых/в работе/закрытых/просроченных, динамика по месяцам,
  Индекс производственной безопасности с приростом к прошлому периоду
- **По разделам:** рейтинг разделов, % нарушений от числа пунктов
- **По регионам:** рейтинг 6 регионов, Индекс ПБ по каждому, топ-5 областей
- **Топ нарушений:** 10 самых повторяющихся нарушений, рейтинг нарушаемых НПА
- Переключатель периода: Месяц / Квартал / Год

### Управление людьми
- Ответственные руководители и инспекторы по БиОТ: добавление, редактирование, удаление
- Поля: Ф.И.О., должность, организация/филиал, телефон, email
- Руководители автоматически появляются в экране «Направить на исполнение»
- Первый инспектор подставляется в форму новой инспекции
- Данные сохраняются в localStorage (не сбрасываются при перезагрузке)
